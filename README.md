# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior to identify sales trends, customer preferences, and business insights using Python, PostgreSQL, SQL, and Power BI. The goal of the project is to transform raw retail data into meaningful business insights through data cleaning, querying, and visualization.

---

# Tools & Technologies Used

- Python (Pandas, NumPy)
- PostgreSQL
- SQL
- Power BI
- Jupyter Notebook

---

# Project Workflow

Raw Dataset  
↓  
Data Cleaning using Python  
↓  
Store Cleaned Data in PostgreSQL  
↓  
Business Analysis using SQL  
↓  
Dashboard Creation in Power BI  

---

# Data Cleaning & Preprocessing

The following preprocessing steps were performed:

- Converted column names into snake_case
- Handled missing values in review ratings
- Created customer age groups
- Grouped payment methods into broader categories
- Renamed columns for better readability
- Exported cleaned dataset to PostgreSQL

---

# Business Questions Analyzed

The project answers several business-related questions, including:

1. Which gender contributes the highest revenue?
2. Which age group spends the most?
3. Which product categories generate the highest sales?
4. How do product preferences vary across age groups?
5. Do subscribed customers contribute more revenue?
6. Does discount usage influence purchase amount?
7. Which products receive the highest customer ratings?

---

# Dashboard Features

The Power BI dashboard includes:

- Total Revenue KPI
- Total Customers KPI
- Average Purchase KPI
- Revenue by Subscription Status
- Product Category Sales by Age Group
- Top Product Categories by Revenue
- Seasonal Revenue Trends
- Interactive slicers for filtering

---

# Key Insight

- Non-subscribed customers contribute the majority of total revenue.
- Clothing is the highest revenue generate category.
- Middle age customer contribute the highest sale.
- Revenue varies significantly across seasons.

---

# Business Recommendations

- Introduce loyalty programs to convert high-value non-subscribed customers into subscribers.
- Focus marketing efforts on high-performing product categories.
- Use age-based targeted promotions for better customer engagement.
- Optimize seasonal inventory and promotional strategies.

This project demonstrates an end-to-end data analytics workflow involving:
